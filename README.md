# RESTfulDC
A RESTful helper for DC RUM.

To use this tool:

1) Copy the .html, .js, and .css files to your CAS in [install]/www_root/custom

2) Navigate to your CAS https://[hostname]/custom/RESTfulDC.html

3) Log in [twice] and start pulling data!
